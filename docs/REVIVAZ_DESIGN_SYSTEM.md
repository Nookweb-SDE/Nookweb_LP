# REVIVAZ — Design System Completo

**SaaS Democrático de Saúde · Versão 1.0**  
Referência oficial para implementação no Cursor AI.

---

## CONCEITO DA MARCA

**REVIVAZ** é uma plataforma de saúde acessível para populações de baixa renda.  
Identidade: **dignidade, acolhimento, tecnologia humana e esperança.**

Referências: [conexasaude.com.br](https://www.conexasaude.com.br) · [gov.br/saude](https://www.gov.br/saude) · [zenklub.com.br](https://zenklub.com.br) · [doctoralia.com.br](https://www.doctoralia.com.br)

## 2. TIPOGRAFIA

- **Display (títulos):** Nunito — 400, 500, 600, 700, 800, 900  
- **Body:** Inter — 400, 500, 600  

Escala: xs 12px · sm 14px · base 16px · lg 18px · xl 20px · 2xl 24px · 3xl 30px · 4xl 36px · 5xl 48px

---

## 3. COMPONENTES

- **Botões:** pill (border-radius full), primary teal-700, secondary green-500, outline/ghost  
- **Cards:** radius-xl (24px), border gray-200, shadow-card  
- **Badges:** teal-100 + teal-700, green-100 + green-700  
- **Navbar:** sticky, blur, height 72px  

---

## 4. SEÇÕES LANDING

1. **Hero:** gradient 135deg teal-700 → teal-300 (ou + green), H1 + body + 2 CTAs + stats  
2. **Trust bar:** logos parceiros  
3. **Como funciona:** 3 steps com ícones  
4. **Planos:** 3 cards (Básico / Familiar / Plus)  
5. **Depoimentos:** carousel, NPS  
6. **Footer:** dark gray-950, 4 colunas
