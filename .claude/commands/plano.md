# /plano — Mostrar plano antes de agir

Antes de executar qualquer alteração, mostre um plano estruturado para o usuário aprovar.

## Formato obrigatório do plano:

```
📋 PLANO DE AÇÃO
━━━━━━━━━━━━━━━━━━━━━━━
🎯 Objetivo: [o que será feito em 1 linha]

📁 Arquivos que serão tocados:
  • [arquivo 1] → [o que muda]
  • [arquivo 2] → [o que muda]

⚠️  Riscos / impactos colaterais:
  • [listar se houver, ou "Nenhum identificado"]

✅ Pronto para executar? Aguardando confirmação.
━━━━━━━━━━━━━━━━━━━━━━━
```

Só avançar depois que o usuário confirmar ("sim", "pode", "vai", "ok" ou similar).
